## 功能提示卡（当前默认交互）

选中 **任意运势功能**（触摸环上图标，或 Boot 短按循环）时，**L0 太极圈内**弹出与 WiFi 连接同款的半透明 **功能提示卡**（`ShowDebugInfo`）。

| 项 | 说明 |
|----|------|
| **标题** | 对应功能的 `func_label`（如「今日运势」），**仅显示 1 行标题** |
| **正文** | 无（短提示卡不显示卦名/宜忌） |
| **时长** | **常驻**，直至电源键短按关闭或切换运势项 |
| **音效** | 无（选中环上图标静音） |
| **电源键短按** | 手动关闭提示卡；切换至其他运势项后再次显示 |
| **Boot 长按** | 仍触发完整运势流程（`ShowFortuneFromMenu`） |

**示例（今日运势）**：

```
今日运势
```

> Boot **长按** 仍使用 `kFortuneMenuFortuneTexts[]` 展示完整卦名与宜忌。

## 子功能说明

### 9 · 学业运势 — 编译开关（计时/架子鼓未编入当前固件）

> **编译开关**：`XIAOZHI_STUDY_SUB_FEATURES=OFF`（`main/CMakeLists.txt`）时，计时器/架子鼓源码保留但不编入固件；学业与其它运势一样仅显示功能提示卡。置 `ON` 并重新编译可恢复 L0 学业子功能区（`STUDY_SUB_FEATURES_ENABLED=1`）。

| 项 | 说明 |
|----|------|
| **触发** | 与其它运势相同：触摸或 Boot 短按选中序号 **9** |
| **界面** | 功能提示卡（见上节），非计时/架子鼓 UI |
| **Boot 长按** | 仍可触发 `fortune.study` 运势流程 |

### 9 · 学业运势 — 专注计时（`STUDY_SUB_FEATURES_ENABLED=1` 时启用）

环上选中 **学业运势**（`FortuneMenuType::Study`，Idle 态）时，**L0 太极圈**（`taiji_container_`）进入 **学业功能区**。

**L0 视觉基线（学业态全程保持）**：

| 保留 | 隐藏 |
|------|------|
| **太极圈鎏金边**（canvas **3px** 外描金环；**始终可见**） | 阴阳鱼 canvas 填色（不绘制黑白鱼） |
| 圈内深色底（与常态阴鱼区一致） | WiFi / BLE **鱼眼** |
| — | 常态 **60s/圈** 旋转（学业态暂停） |

> 即：选中学业后，用户看到的是 **「只有金色圈的空太极盘」** + 功能图标 / 计时 UI，**不是**完整太极鱼。

#### 状态一览

| 状态 | L0 显示 | 进入方式 | 退出方式 |
|------|---------|----------|----------|
| **学业功能区** | **鎏金圈** + 计时器 / 架子鼓（**默认选中计时**） | 环上选中序号 **9** | 切换环上其他功能 → 恢复太极鱼 + 鱼眼 |
| **专注计时中** | **鎏金圈** + **1 分钟**倒计时 + **内部计时环** | 功能区 **计时已选中** 时 **Boot 短按** | **电源键**短按 → 恢复完整太极圈；或 **60s** 结束 |
| **计时结束 BGM** | **鎏金圈** + `00:00` + 计时环归零 | 60s 走完 | **约 5s** 后自动回功能区；电源键可退出学业区 |
| **计时结束** | 同 BGM 态 | — | BGM 播完 → 功能区（仍保留鎏金圈，**默认仍选中计时**） |

#### 学业功能区（Menu · 子态）

| 项 | 说明 |
|----|------|
| **触发条件** | Idle 态下触摸扇区或 Boot 短按选中序号 **9** |
| **默认子项** | **`StudyMenuItem::Timer`（计时）**；进入时自动高亮时钟图标 |
| **子项切换** | 点击区内 **时钟** / **架子鼓** 图标；选中音 `OGG_POPUP` |
| **区内图标尺寸** | 比外围运势环 **大 50%**（35px → **53px** 视觉）；选中 **1.1×** 高亮 |
| **区域** | `taiji_container_` 全径；进入学业态时 L0 **整体顺时针旋转 90°**（鎏金圈 + 区内图标/计时环） |
| **Boot 短按** | 仅当 **计时** 选中时启动 **专注计时** |
| **Boot 长按** | 学业态下 **不**调用 `ShowFortuneFromMenu()` |

**学业功能区图标**：

| 元素 | Font Awesome 宏 | 语义 | 视觉约 | 备注 |
|------|-----------------|------|--------|------|
| 计时器 | `FONT_AWESOME_CLOCK` | 专注计时（**默认选中**） | **53px** | Boot 短按启动 1 分钟计时 |
| 架子鼓 | `FONT_AWESOME_DRUM` | 节奏练习 | **53px** | 字库已扩展；显示真鼓图标 🥁 (U+F569) |

**布局示意**（Menu 态，`taiji_container_` 内）：

```
        ╭─────────╮
       ╱  鎏金圈   ╲    ← 3px 金边，无阴阳鱼
      │             │
      │  [CLOCK]*   │    ← 默认选中，53px
      │   [DRUM]    │    ← 53px
      │             │
       ╲           ╱
        ╰─────────╯
```

#### 专注计时中（FocusRunning · 子态）

**Boot 短按**（计时已选中）后，在 **保留鎏金圈** 的前提下进入 60s 专注倒计时。

| 项 | 说明 |
|----|------|
| **时长** | 固定 **60s**（`STUDY_FOCUS_DURATION_MS = 60000`） |
| **鎏金圈** | **外层太极金圈始终保留**（与 Menu 态相同） |
| **内部计时环** | `study_focus_arc_`，外缘半径约 **r=78**（在鎏金环内缘留 **5px** 间距，不触及太极环） |
| **环宽 / 配色** | 底轨 **24px** 深灰 `#3A3A3A`；进度条 **27px** 青色 `#4FC3F7`（较初版各 **+50%**） |
| **中心** | 剩余时间 `mm:ss`（`01:00` → `00:00`） |
| **电源键（GPIO7）** | **短按** → 退出学业区，恢复 **完整太极鱼** + 鱼眼；环上仍高亮学业项，需再次选中序号 9 或 Boot 切回才可重入 |
| **计时结束** | `OGG_POPUP` 提示音 → **CompleteBgm** 播放 **5s** `OGG_STUDY_FOCUS_BGM` |

**布局示意**（FocusRunning 态）：

```
        ╭─────────╮
       ╱  鎏金圈   ╲    ← 外层：始终可见
      │  ╭─────╮  │
      │ ╱计时环╲ │    ← 24px 灰轨 + 27px 青进度（外缘距金环 5px）
      │ │01:00 │ │
      │ ╲     ╱  │
      │  ╰─────╯  │
       ╲           ╱
        ╰─────────╯
```

#### 音频

| 事件 | 音效 | 路径 |
|------|------|------|
| 子项切换 | 清脆提示音 | `PlayUiSound(OGG_POPUP)` |
| 60s 计时自然结束 | 提示音 + **约 5s 背景音乐** | `OGG_POPUP` → `OGG_STUDY_FOCUS_BGM` |
| 资源文件 | `main/assets/common/study_focus_bgm.ogg` | `scripts/gen_study_focus_bgm.py` 生成 |

#### 交互流程

```
环上选中「学业运势」(9)
        │
        ▼
  鎏金圈 + [CLOCK]* + [DRUM]     ← 默认选中计时
        │
        │ Boot 短按（计时选中）
        ▼
  鎏金圈 + 内部计时环 60s
        │
        ├─ 电源键短按 ──────────► 完整太极圈（退出学业区）
        │
        └─ 60s 结束 ───────────► POPUP + 5s BGM ─► 功能区
```
