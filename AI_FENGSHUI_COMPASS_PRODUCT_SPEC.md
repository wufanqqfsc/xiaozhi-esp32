**文档版本**: v2.1  
**生成日期**: 2026年6月  
**适用平台**: Waveshare ESP32-S3-Touch-LCD-1.85B

## 一、产品定义

### 1.1 产品名称
**小智AI姿态平衡仪** - 融合传统平衡智慧与现代AI技术的智能交互设备

### 1.2 产品定位
- **目标用户**: 追求生活品质人群、家居装饰爱好者、传统文化爱好者、产品设计师
- **核心价值**: 提供直观的设备姿态信息，通过AI解读"平衡"与"和谐"的生活美学
- **产品口号**: "智掌平衡，慧启和谐"

### 1.3 产品理念

结合中国传统风水文化中的"平衡"概念：
- **阴阳平衡**: 设备水平代表阴阳和谐
- **五行平衡**: 倾斜状态对应五行相生相克
- **动静平衡**: 旋转速度代表动态平衡
- **方位感应**: 通过设备朝向（需配合手机APP）获取方位信息

### 1.4 产品特性

| 特性 | 描述 |
|------|------|
| **智能语音** | 支持语音唤醒和语音交互 |
| **水平检测** | 实时显示设备是否处于水平状态 |
| **姿态显示** | 显示俯仰角、翻滚角、偏航角 |
| **动态感知** | 显示旋转角速度和累积旋转角度 |
| **AI解读** | 基于姿态状态给出"平衡"与"和谐"的解读 |
| **炫酷界面** | 圆形屏幕展示动态气泡水平仪效果 |
| **氛围灯光** | LED颜色根据倾斜程度变化 |

## 二、功能需求分析

### 2.1 核心功能模块

#### 模块一：水平检测显示
- 实时显示设备X轴（俯仰）和Y轴（翻滚）倾斜角度
- 动态气泡水平仪可视化效果
- 水平状态指示（水平/倾斜）
- 角度精确数值显示（保留1位小数）

#### 模块二：姿态角显示
- **俯仰角 (Pitch)**: 前后倾斜角度，范围 -180° ~ +180°
- **翻滚角 (Roll)**: 左右倾斜角度，范围 -90° ~ +90°
- **偏航角 (Yaw)**: 水平旋转角度（相对初始位置），范围 0° ~ 360°
- 角度单位显示（度°）

#### 模块三：动态旋转感知
- 实时旋转角速度（°/s）
- 累积旋转角度
- 旋转方向指示（顺时针/逆时针）
- 旋转状态判断（静止/慢速/快速）

#### 模块四：AI平衡解读
- 基于当前倾斜角度的"五行平衡"解读
- 水平状态时的"阴阳和谐"解读
- 旋转状态时的"动态平衡"解读
- 生活场景建议（摆放位置、调整方向）

#### 模块五：语音交互
- 语音唤醒（"小智你好"）
- 语音查询当前姿态状态
- 语音播报平衡解读
- 支持简单问答

#### 模块六：氛围效果
- 设备水平时：绿色系灯光
- 轻微倾斜：黄色系灯光
- 严重倾斜：红色系灯光
- 语音交互时的动态灯光效果

### 2.2 交互场景

**场景一：日常水平检测**
1. 用户说"小智你好，现在水平吗？"
2. 设备唤醒并读取IMU数据
3. 屏幕显示气泡水平仪和角度数值
4. 语音播报"当前设备基本水平，倾斜角度约2度"

**场景二：调整物品水平**
1. 用户将设备放在待测物品上
2. 观察气泡位置和颜色
3. 调整物品直到气泡居中（绿色）
4. 语音播报"已水平！"

**场景三：姿态查询**
1. 用户说"小智你好，现在倾斜多少？"
2. 设备读取并显示三个轴的角度
3. 屏幕显示具体数值
4. AI给出调整建议

**场景四：旋转检测**
1. 用户说"小智你好，我在转圈"
2. 设备显示当前旋转速度和累积角度
3. 实时更新旋转数据
4. 播报"您已旋转180度"

**场景五：平衡知识问答**
1. 用户说"小智你好，什么是阴阳平衡？"
2. 设备通过LLM生成回答
3. 屏幕显示文字内容
4. 语音播报详细解释

### 2.3 非功能需求

| 需求类型 | 要求 |
|---------|------|
| **响应时间** | 唤醒响应 < 500ms |
| **角度精度** | ±1°（静态） |
| **显示刷新** | 30fps |
| **语音识别** | 准确率 > 95% |
| **待机时间** | > 6小时（锂电池） |
| **网络依赖** | 支持离线基础功能，联网增强体验 |

#### 主界面：圆形气泡水平仪设计

**核心元素**:
1. **圆形装饰边框**: 传统罗盘风格的同心圆装饰（半径100、110、120像素）
2. **十字刻度**: 水平和垂直方向的刻度线（长度15像素）
3. **方向标记**: 东南西北四个方向文字（距离中心105像素）
4. **气泡球**: 动态气泡（直径20像素），跟随设备倾斜移动
5. **中心标记**: 十字准星（直径40像素）
6. **发光效果**: 气泡和刻度线带有发光效果

**视觉风格**:
- 深色渐变背景（从中心向外由深蓝到深灰）
- 金色刻度线和边框装饰
- 气泡带有光晕效果
- 倾斜时气泡平滑移动动画（使用LVGL动画框架）

**气泡移动规则**:
```
倾角 0°   → 气泡居中（绿色，发光半径8像素）
倾角 1-3° → 气泡轻微偏移（浅绿色，发光半径6像素）
倾角 3-5° → 气泡中等偏移（黄色，发光半径5像素）
倾角 >5°  → 气泡明显偏移（红色，发光半径4像素）
```

**气泡位置计算公式**:
```
气泡X偏移 = 倾斜角(度) × 灵敏度系数
气泡Y偏移 = 倾斜角(度) × 灵敏度系数

灵敏度系数 = 80像素 / 30度 ≈ 2.67像素/度
```

### 4.4 传感器融合算法

由于QMI8658只有陀螺仪和加速度计（无磁力计），偏航角（Yaw）会有漂移。

#### 加速度计计算俯仰角和翻滚角

```
Pitch = atan2(accel_y, sqrt(accel_x² + accel_z²))
Roll = atan2(-accel_x, accel_z)
```

优点：静态精度高
缺点：动态运动时有噪声

#### 陀螺仪积分计算偏航角

```
Yaw += gyro_z * dt
```

优点：动态响应好
缺点：积分漂移，长时间累积误差

#### 数据融合方案

使用互补滤波融合两种数据源：

```
// 互补滤波示例
pitch = 0.98 * (pitch + gyro_x * dt) + 0.02 * accel_pitch
roll = 0.98 * (roll + gyro_y * dt) + 0.02 * accel_roll
yaw = yaw + gyro_z * dt  // 偏航角仅用陀螺仪
```

### 4.5 状态机设计

#### 设备状态

| 状态 | 描述 | 触发条件 |
|------|------|---------|
| **待机** | 设备静止，显示水平仪 | 系统启动完成 |
| **唤醒** | 检测到唤醒词 | 语音唤醒/触摸唤醒 |
| **听** | 正在接收语音输入 | 唤醒成功 |
| **思考** | LLM处理中 | 语音输入完成 |
| **说** | 正在播放回答 | LLM返回结果 |
| **校准** | 正在校准IMU | 校准指令 |
| **设置** | 进入设置界面 | 长按屏幕 |

#### 状态转换

```
待机 → 唤醒 → 听 → 思考 → 说 → 待机
              ↓
            校准 → 待机
              ↓
            设置 → 待机
```

| 向左倾斜 | 木 | 生长发展，需防过盛 |
| 向右倾斜 | 金 | 收敛凝聚，注意调整 |
| 严重倾斜 | 土（失衡） | 需要调整，恢复平衡 |

## 八、风险评估

### 8.1 技术风险

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| 偏航角漂移 | 中 | 陀螺仪积分导致的偏航角长期漂移 | 显示相对角度，定期重置 |
| 动态精度 | 中 | 运动中加速度计数据不可靠 | 使用滤波算法，动态调整权重 |
| 温度漂移 | 低 | IMU随温度变化有微小漂移 | 软件校准补偿 |

### 8.2 进度风险

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| 算法调试 | 中 | 传感器融合需要调参 | 预留调试时间 |
| UI优化 | 低 | 动画效果需要迭代 | 多轮测试反馈 |

### 8.3 使用限制说明

**重要提醒**:

本期产品使用板载QMI8658 IMU（无磁力计），存在以下限制：

| 功能 | 状态 | 说明 |
|------|------|------|
| **俯仰角(Pitch)** | ✅ 完全支持 | 静态精度±1° |
| **翻滚角(Roll)** | ✅ 完全支持 | 静态精度±1° |
| **偏航角(Yaw)** | ⚠️ 有限支持 | 有漂移，适合短时相对测量 |
| **地理方向** | ❌ 不支持 | 无磁力计，无法感知东南西北 |

**偏航角漂移说明**:
- 静止状态下，偏航角几乎不漂移
- 长时间旋转后，偏航角可能累积几度到几十度误差
- 如需精确偏航角，可后续外接QMC5883L磁力计
